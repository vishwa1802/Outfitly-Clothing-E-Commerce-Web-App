# 👕 Outfitly – Clothing E-Commerce Web App

Outfitly is a modern clothing e-commerce web application designed to provide a smooth and interactive online shopping experience. Users can browse clothing products, view detailed product information, add items to their cart or wishlist, and place orders easily.

The application is built using **React.js and Tailwind CSS**, focusing on responsive design and reusable component architecture. It demonstrates practical implementation of modern front-end development concepts.

---

# 🚀 Features

* 🛍 Browse clothing products
* 🔎 View detailed product information
* 🛒 Add and remove items from cart
* ❤️ Wishlist functionality
* 👤 User authentication (Login & Signup)
* 📦 Order management
* 📍 Address management
* 🛠 Admin panel for product management
* 📱 Fully responsive UI design

# ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/yourusername/outfitly-clothing-app.git
```

2. Go to project folder

```
cd outfitly-clothing-app
```

3. Install dependencies

```
npm install
```

4. Run the project

```
npm run dev
```
